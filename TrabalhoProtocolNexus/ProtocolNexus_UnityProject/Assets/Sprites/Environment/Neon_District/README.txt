HOW TO USE
----------
  Unity  : Import sprites, set Filter Mode to Point, PPU to 16.
  Godot  : Import as 2D Pixel Art, Nearest filter, Mipmaps off.
  NPC    : Spritesheet is 3 columns x 4 rows (32x48 px per frame).
           Rows = South / West / East / North.
           Columns = Idle / Step-Left / Step-Right.
